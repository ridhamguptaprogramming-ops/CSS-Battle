// what is it?

The Ultimate CSS Battle

Participants recreate pixel-perfect designs using only pure HTML & CSS — no JavaScript, no libraries, no shortcuts. Speed, precision, and elegance decide the winner.

🎨

Creative Thinking

Translate complex visual designs into clean, minimal CSS code.

⚡

Frontend Mastery

Demonstrate deep command of selectors, properties, and layout systems.

🔧

Code Optimization

Fewer characters, higher scores. Write lean, precise, expressive CSS.

🧠

Problem Solving

Overcome layout puzzles under time pressure with clever solutions.

// scoring system

Judging Criteria

Three dimensions determine your score. Precision wins, brevity earns bonus, and speed seals the deal.

60%
Visual Similarity
How closely your output matches the target design — pixel by pixel, color by color.
25%
Code Length
Leaner code scores higher. The elegance of your solution matters as much as correctness.
15%
Submission Speed
Faster submissions earn extra points. Quality first — but swift wins break ties.
